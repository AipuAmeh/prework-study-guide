HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Section elements are themantic elements that define a section in a document that contains themantic content.
  • You must save information often!

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The period preceding the "card" designates it as a class selector.
  • Any element assigned to card will inherit the same properties.
  • You can assign colors using semantic elements, or through a hexadecimal code.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: navigates to main branch.
  • git stash: stashes/saves changes.
  • git add -A: stages changes that were made.
  • git commit -m: commits changes to repository with a message.
  • git pull origin main: makes sure our branch is up to date before opening a pull request.
  • git push origin branch-name: pushes changes up to repository.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script. That is from top to bottom.
  • In order to execute a function, you must call it first.
  • Conditional statements can be used to interrupt control flow.
  • An array is a single variable used to hold a group of data that are related in some way.
  • A for loop allows a single block of code to be executed over and over.
  • The first statement of a loop determines the starting point.
  • The second statement of a loop is the condition.
  • The third statement of a loop allows the array to iterate over each item.
